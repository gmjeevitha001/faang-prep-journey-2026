# Time & Space Complexity

## Key Rules
- Always consider worst case
- Ignore constants
- Ignore less dominant terms

## Common Complexities
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ)

## Nested Loops
- n × n → O(n²)
- Varying limits → analyze total iterations

## Space Complexity
- Input space + Auxiliary space
- Output array counts as space
